Semana 12 · Modelado 3D de la Carcasa del Prototipo

Diseño · Espacios internos · Aberturas · Ensamble · Iteración

Introducción

Durante esta sesión se desarrolló la primera versión completa del modelado 3D de la carcasa del dispositivo PostureAlert. El enfoque estuvo orientado a crear un contenedor funcional que permita alojar todos los componentes electrónicos necesarios, garantizando orden interno, accesibilidad y una estructura adecuada para impresión 3D.

Volumen Principal del Diseño

En esta sección se definió la forma base del prototipo, considerando factores como:

  • Comodidad para el usuario.
  • Estética acorde al propósito ergonómico.
  • Estabilidad durante el uso.
  • Compatibilidad con impresión 3D (grosor, paredes, soportes).

Espacios Internos para la Electrónica

Se incorporaron cavidades internas destinadas a alojar los componentes del sistema:

  • Microcontrolador (Arduino Nano o similar).
  • Batería o sistema de alimentación.
  • Canales internos para cableado.
  • Compartimientos para sensores o módulos adicionales.

Aberturas Exteriores

Se añadieron cortes y perforaciones necesarias para la interacción del usuario y el correcto funcionamiento del circuito:

  • Entrada para USB o puerto de alimentación.
  • Orificios destinados a sensores (ToF, ultrasonido, micrófono, etc.).
  • Ranuras de ventilación para disipación de calor.
  • Agujeros guía para fijación o correas.

Sistema de Cierre

Para asegurar el ensamblaje correcto del dispositivo, se evaluaron distintos métodos de cierre:

  • Encaste o snap-fit (sin tornillos).
  • Tapa a presión.
  • Uso de tornillos para mayor fijación si fuera necesario.

Entregables

  • Archivo editable del modelado 3D (.f3d, .sldprt, .blend, .skp, etc.).
  • Archivo exportado en .STL listo para impresión.
  • Imágenes del proceso de diseño en 4 etapas:
    • Boceto inicial.
    • Modelado base sin detalles.
    • Aberturas y cavidades para electrónica.
    • Versión refinada final.
  • Diagrama de ubicación interna de componentes.
  • Reflexión escrita sobre cambios realizados.

Reflexión Final

En esta semana se consolidó la base estructural del producto, permitiendo que el prototipo avance hacia su primera versión completamente funcional. El proceso evidenció la importancia del espacio interno, la organización del circuito y la necesidad de realizar iteraciones constantes para garantizar precisión y comodidad para el usuario.
